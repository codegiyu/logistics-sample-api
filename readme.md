# 📁 Collection: WEBSOCKET

## Client Connection Note
Token should be passed from client when connecting to server. If token is not present or invalid, the connection will be terminated.

## Client Connection Sample
>```
socket = io.connect("http://localhost:4000", {
    query: {
        token: `your-token-here`
    }
});
>```
